Mise à jour portfolio Rudy Gaillot

Contenu :
- index.html mis à jour avec la SAE Régulation LTG et les SAE 0 à 5.
- sae.html mis à jour avec les cartes des nouvelles SAE et les liens PDF.
- documents.html mis à jour avec les PDF corrigés.
- PDF corrigés avec noms simples.
- Images d'aperçu générées depuis les PDF.

Publication :
1) Décompresser ce ZIP.
2) Ouvrir le dossier décompressé.
3) Envoyer TOUT le contenu à la racine du dépôt GitHub Portofolio-Rudy.
4) Dans GitHub Pages, garder : Branch = main, Folder = /root.
5) Attendre 1-2 minutes, puis faire Ctrl+F5 sur le site.

Important :
- Ne mets pas le dossier lui-même dans GitHub, mets son contenu.
- Les liens sont relatifs : index.html, style.css, main.js, fichiers PDF et images doivent être au même niveau.
