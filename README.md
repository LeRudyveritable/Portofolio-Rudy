# Portfolio Rudy Gaillot - GitHub Pages

Site statique multi-pages pour un portfolio BUT GEII parcours Électronique et Systèmes Embarqués.

## Structure

- `index.html` : page d'accueil
- `competences.html` : compétences GEII
- `sae.html` : réalisations SAE et projets
- `stages.html` : stages et expériences professionnelles
- `documents.html` : CV, rapports et documents PDF
- `assets/css/style.css` : design du site
- `assets/js/main.js` : menu mobile et année automatique
- `documents/` : dossier où déposer les PDF

## PDF à ajouter dans le dossier documents

Pour que les boutons fonctionnent, ajoute les fichiers PDF avec ces noms :

- `cv-rudy-gaillot-fr.pdf`
- `cv-rudy-gaillot-en.pdf`
- `fiche-competences.pdf`
- `rapport-stage-but3-aube-electronique.pdf`
- `rapport-stage-but2-aube-electronique.pdf`
- `dossier-sae.pdf`

Tu peux modifier les noms des liens dans `documents.html` si besoin.

## Publication GitHub Pages

Le dépôt s'appelle `Portofolio-Rudy`, donc l'adresse du site est normalement :

`https://lerudyveritable.github.io/Portofolio-Rudy/`

Dans GitHub : `Settings` -> `Pages` -> `Deploy from a branch` -> branche `main` -> dossier `/ root`.
