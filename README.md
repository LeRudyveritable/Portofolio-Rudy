# Portfolio Rudy Gaillot

Portfolio personnel de Rudy Gaillot, étudiant en BUT GEII parcours Électronique et Systèmes Embarqués.

Le site est un portfolio statique multi-pages :

- `index.html` : accueil
- `competences.html` : compétences GEII et apprentissages critiques
- `sae.html` : SAE et projets par année
- `stages.html` : stages et expériences
- `documents.html` : documents PDF

Publication GitHub Pages : branche `main`, dossier `/root`.
