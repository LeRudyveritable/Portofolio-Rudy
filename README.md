# Portfolio Rudy Gaillot - BUT GEII

Site statique prêt pour GitHub Pages.

## Mise en ligne

1. Décompresser ce ZIP.
2. Ouvrir le dépôt GitHub `Portofolio-Rudy`.
3. Envoyer tout le contenu du dossier décompressé à la racine du dépôt.
4. Faire `Commit changes`.

Adresse du site :

`https://lerudyveritable.github.io/Portofolio-Rudy/`

## Structure

- `index.html` : accueil
- `competences.html` : compétences GEII
- `sae.html` : SAE et projets
- `stages.html` : stages et expériences
- `documents.html` : CV, rapports, fiche compétences, dossiers SAE
- `assets/css/style.css` : design du site
- `assets/js/main.js` : menu mobile, filtres, animations
- `assets/img/previews/` : aperçus visuels extraits des PDF
- `assets/img/sae/` : emplacement prévu pour tes vraies photos SAE
- `documents/` : PDF propres et renommés

## Important

`index.html` doit rester directement à la racine du dépôt, pas dans un sous-dossier.
